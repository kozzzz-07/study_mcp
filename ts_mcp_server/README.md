# btc_mcp_server

By using this MPC, you can obtain the latest BTC price.  
After adding the MCP, try asking questions about the BTC price.  

## settings

your MCP client settings json.

```json
{
  "mcpServers":{
    "btc-binance-mpc": {
      "command": "/npx",
      "args": ["btc-binance-mpc"]
    }
  }
}
```
